При изменении местонахождения объекта НВОС эколог действует по цепочке: актуализирует сведения в реестре → переоформляет КЭР/ДВОС → обновляет проекты → корректирует СЗЗ.

  • Актуализация — обязательна при изменении фактического адреса (30 дней)
  • Переоформление — требуется для КЭР, ДВОС, проектов НДВ/НДС
  • Корректировка — нужна для СЗЗ при изменении границ воздействия

Важно: Устав организации и другие учредительные документы не меняются, если изменилось только местонахождение объекта, а не юридический адрес.
