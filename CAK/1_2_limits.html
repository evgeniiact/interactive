Отберите вещества для проверки по источнику

На каждом экране один вопрос. Посмотрите документы справа и отметьте вещества, которые нужно взять в дальнейшую проверку по САК.

Кейс 1 из 3
Что вы должны запомнить

Сначала определите, по какой нормативной ветке идет объект. Потом возьмите только те вещества, которые относятся к этой ветке. И только после этого сравните их с порогами.

В дальнейшую проверку по САК попадает не любое вещество из инвентаризации, а только то, которое соответствует нормативной ветке объекта, есть в выбросе и превышает порог по массе.

Мини-результат

После этого интерактива вы умеете отбирать вещества по источнику с учетом нормативной ветки и порогов. Это ключевой шаг перед расчетом ИЗАВ и проектированием САК.
